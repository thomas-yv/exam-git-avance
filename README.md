# Bibliothèque numérique

Ceci est un projet de site web qui liste des livres défini aupréalable avec leur titre, leurs 
auteurs et l'année de sortie. Vous pouvez avoir plus d'informations 
sur un livre en particulier en cliquant sur "Voir le livre".

## Initialisation

Pour démarrer ce projet, vous devrez suivre les étapes ci-dessous :
GO doit être installé sur votre machine : [Install GO](https://golang.org/doc/install)

```bash
$ git clone https://github.com/thomas-yv/exam-git-avance
$ cd exam-git-avance
$ go mod init main
```

## Lancer le serveur web
```bash
$ go run .
```

La bibliothèque sera accessible sur ce lien : [Lien vers la bibliothèque](http://localhost:8080).
